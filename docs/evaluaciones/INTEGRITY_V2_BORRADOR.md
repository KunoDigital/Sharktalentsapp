# Integridad V2 — Borrador de 90 preguntas

## Formato
- Cada pregunta tiene 4 opciones (A, B, C, D)
- risk_weights: 0 = sin riesgo, 1 = riesgo bajo, 2 = riesgo medio, 3 = riesgo alto
- Las opciones NO están ordenadas de mejor a peor (se mezclan)
- Mezcla de formatos: situacional, opinión, admisión, frecuencia
- Las preguntas de Buena Impresión (BI) están mezcladas entre las demás

## Dimensiones
- **ÉTICA**: Autenticidad, Inteligencia social, Imparcialidad, Sencillez, Dominio personal
- **RIESGO**: Honestidad, Hurto, Soborno, Alcohol, Drogas, Apuestas, Confiabilidad
- **BI**: Buena Impresión (detector de mentiras)

## NOTAS DE IMPLEMENTACIÓN

### Orden de presentación
Las preguntas NO se presentan agrupadas por dimensión. Se mezclan aleatoriamente para que el candidato no detecte patrones. Las preguntas de BI se intercalan entre las demás.

### Shuffling de opciones
Las opciones se barajan (Fisher-Yates) para cada candidato, como ya hacemos.

### Scoring de Buena Impresión
Se calcula como una dimensión separada. Si el score de BI es alto (>60%), se marca como "Posible deseabilidad social exagerada" y se recomienda profundizar en entrevista. NO invalida el test pero es una bandera.

### Tiempos de respuesta
Registrar tiempo por pregunta. Tiempos muy largos en preguntas de riesgo pueden indicar que el candidato está pensando demasiado su respuesta.

### Umbrales
- Bajo: 0-30% de riesgo por dimensión
- Medio: 31-60%
- Alto: >60%
- BI: >60% = "Respuestas posiblemente exageradas"
